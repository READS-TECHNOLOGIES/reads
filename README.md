## Anti-Cheat System (Token Farming Protection)

To protect the platform and token economy, $READS includes **multi-layer anti-cheat logic** designed specifically to prevent **token farming**.

### 1. Admin-Controlled Quiz Configuration

Admins define:
- Question pool size (e.g. 10 questions)
- Questions shown per attempt (e.g. 3 random)
- Token reward amount (50, 100, 200 tokens)
- Minimum passing score (70%, 80%, etc.)
- Cooldown time between attempts

This prevents uncontrolled reward distribution.
